## CHAPTER01 컴퓨터는 데이터에서 배운다

**머신 러닝 종류**
1. 지도 학습
   - 레이블된 훈련 데이터에서 모델을 학습하여 본 적 없는 미래 데이터에 대해 예측
   - 학습 데이터가 없다면 그에 대한 예측이 이루어질 수 없음
   1. 분류(Classification) : 클래스 레이블 예측
      - 이진 분류 : 두 개의 클래스 사이에서 구분
      - 다중 분류 : 3개 이상의 클래스 사이에서 구분
      - 결정 경계 : 클래스를 나누는 기준
   2. 회귀(Regression) : 연속적인 출력 값 예측
      - 예측 변수와 반응 변수를 통해 출력값 예측
2. 비지도 학습
   - 레이블되지 않거나 구조를 알 수 없는 데이터를 다룸
   1. 군집(Clustering) : 서브그룹 찾기
      - 각 클러스터는 어느 정도 유사성을 공유하고 다른 클러스터와 비슷하지 않은 샘플 그룹 형성
      - 정보를 조직화하고 데이터에서 의미 있는 관계 유도
   2. 차원 축소(Dimensionality Reduction) : 데이터 압축
      - 하나의 관점 샘플에 많은 측정 지표가 있는 경우 사용
      - 잡음(Noise) 데이터를 제거하는 과정으로 인해 특정 알고리즘 예측 성능 감소
3. 강화 학습
   - 환경과 상호 작용하여 시스템 성능 향상이 목적
   - 보상 신호는 레이블 값이 아니라 행동에 대한 점수

이 책에선 분류, 회귀, 군집을 다룸

**전처리 : 데이터 형태 갖추기**
- 원본 데이터로는 학습 알고리즘이 최적의 성능을 내기는 어려움
- 최적의 성능을 내려면 선택된 특성이 같은 스케일을 가져야함
- 특성을 [0, 1] 범위로 변환하거나 평균이 0이고 단위 분산을 가진 표준 정규 분포로 변환
- 높은 상관관계로 인해 중복된 데이터는 차원 축소 기법을 이용하여 저차원 부분 공간으로 압축
- 데이터셋을 랜덤하게 훈련 세트와 테스트 세트로 나누어서 수행
  - 훈련 세트 : 모델 훈련(가중치 찾기)에 쓰는 데이터 세트
  - 테스트 세트 : 모델의 실제 정확도를 판별하기 위해 쓰는 데이터 세트

**예측 모델 훈련과 선택**
- 각 알고리즘 마다 편향성이 있으므로 특정 알고리즘이 우월하다고 할 수 없음
- 여러가지 알고리즘을 사용해보고 분류의 경우 정확도를 기준으로 모델의 성능을 측정해야함
